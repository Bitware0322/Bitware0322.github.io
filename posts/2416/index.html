博途安装时各种报错，各式各样的报错估摸有10种了吧，所以开个文章记录一下。

注：该文章主要内容来自

收藏，西门子博途编程软件安装失败常见问题汇总（附软件安装视频） - 哔哩哔哩

各类报错汇总和解决方法

1、必须重新启动计算机，然后才能运行安装程序，要立即重启计算机吗？

**处理方法：**键盘按下Win+R，输入regedit打开注册表编辑器，依次打开HKEY_LOCAL_MACHINE>SYSTEM>CurrentControlSet>Control>Session Manager找到PendingFileRenameOperations选项，然后右键删除。若安装过程中反复出现，需要反复删除。

2、应用程序无法启动，因为应用程序的并行配置不正确。

处理方法： 这个作者目前没遇到过，根据网上方法，最好的方式就是重装系统win10专业版来的最快，对电脑不太懂的人，短时间内很难有解决。

3.打开博途软件乱码。

处理方法： 升级您的电脑IE到11以上版本就可以解决，原因是操作系统的IE浏览器版本太低

4.TIA portal安装需要.NET3.5 SP1.请在PC中启动.NET 3.5 SP1

处理方法： 控制面板–程序和功能–添加或关闭WINDOWS功能–打开的对话框里面勾选NET3.5，然后等待下载安装完成就行了。或者网上下载一个NET3.5安装包直接安装即可。

5. 博途V14wincc升级到sp1时，提示要安装KB3033929，但是KB3033929一直安装不上，有没有解决的办法

处理方法： 这个一般在WIN7系统出现，需要先安装KB3033929安全补丁，这个补丁在网上搜索就能找到。安装完补丁后需要在控制面板中安装IIS服务

打开控制面板，在“程序和功能”中点击“打开或关闭Windows功能”，然后勾选“Internet信息服务”，点击按钮“确定”即可。

6.由于缺少所需的产品，对项目的应用会受到限制

处理方法： 此错误指的是你打开的程序不是当前博途的版本，请安装相对应的版本软件再打开程序。

7，打开软件或者操作过程中出现TIA Portal 博途出错，需要关闭

处理方式： 此错误最好的办法就是重装系统。或者完全卸载当前博途软件重装一次

8.找不到许可证“”step7 professional 或者 basic等“”

处理方式： 这种一般是Automation License Manager的服务没启动造成的，键盘按下Win+R 后输入SERVICES.MSC，找到Automation License Manager Service右键启动。如果没找到这个服务，需要重新安装Automation License Manager，然后重新安装密钥

9.操作系统需要使用基于美国联邦信息处理标准的加密算法

**处理方式：**此问题，典型的操作系统问题，更换系统

10.安装过程中出错

处理方式： 新电脑第一次装一般有这样的结果，关闭后把电脑系统修复一次再进行重新装即可，如果还不能成功只有重装更换系统速度最快

11.许可证传送

**处理方式：**直接跳过许可证，查看安装视频有授权说明和视频

12.V13 SP1仿真出现此问题

处理方法：

下载这个文件安装上

链接：https://pan.baidu.com/s/1X7TdWmk75VBI9tcloZ68JA

密码：whxr

13.出现下面的问题，安装时候部分文件已损坏。

**处理方式：**此问题由于下载过程中出现中断，或者电脑中安装杀毒软件和360安全卫士等造成认为博途软件是病毒文件误删除。博途安装文件不完整或者丢失导致

14.Microsoft visual C++ 2005问题

Command line option syntax error type command /for help

处理方式： 该原因是由于你的用户名是中文造成的，你可以选择确定继续安装，对后面软件的使用没影响。如果你不想让他出现，你需要在网上查找方法将中文用户名改成英文用户名。

15.STEP7 Professional的许可证无法彻底完成

**处理方式：**这个是因为你Automation License Manager的版本低了，跟系统不兼容，安装Automation License ManagerV6.2后，然后重新授权一遍即可解决。

16.S7-PLCSIMV16签名无效。

处理方试：

  • 1.重新授权博途
  • 2.重新安装仿真软件（即SIMATIC S7-PLCSIM）
